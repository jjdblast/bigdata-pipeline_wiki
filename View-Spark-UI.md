### Apache Spark Driver Admin Web UI
* Jobs, Stages, Tasks
* Environment Variables
* Event Timeline
* Streaming Tab
* If multiple jobs are actively running, the port will start at `34040` and +1 per active job
```
macosx-laptop$ open http://127.0.0.1:34040
```
